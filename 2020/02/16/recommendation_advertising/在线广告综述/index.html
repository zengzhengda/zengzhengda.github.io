在线广告综述

发表于 2020-02-16 | 分类于 计算广告 | 0 comments |

广告的定义与目的

定义

  • 要素

    • 出资人

      • 广告主
      • 代理商
      • 采买商

    • 媒体

    • 受众

  • 广告分类

    • 品牌广告

      • 品牌宣传
      • 中长期购买率和利润空间

    • 效果广告

      • 马上购买和其他转化行为

      • 数字媒体的出现，蓬勃发展

        • 低成本地投放个性化广告
        • 更清晰了解用户意图

目的

  • 广告主通过媒体达到低成本的用户接触

在线广告创意类型

产品形态

用户接触的创意形式

  • 横幅广告
  • 文字链广告
  • 富媒体广告

  • 视频广告

    • 前插片
    • 后插片
    • 暂停

  • 社交广告

  • 移动广告

    • 横屏
    • 开屏
    • 插屏
    • 积分墙
    • 推荐墙

  • 邮件定向营销广告

    • 短信、imessage等也类似

在线广告简史

展示广告

  • 售卖方式

    • 合约广告

  • 问题: 当媒体的流量和品牌认知度都相对稳定以后，又有什么新的提高收入的办法呢？

    • 个性化广告

定向广告

  • 定向

    • 受众定向
    • 广告投放

  • 售卖方式

    • 担保式投送

  • 在线分配

    • 一是如何有效地将流量分配到各个合约互相交叉的人群覆盖上
    • 二是要在在线的环境下实时地完成每一次展示决策。

竞价广告

  • 动机

    • 很难对这些细粒度标签组合的流量做准确预估
    • 当一次展示同时满足多个合约的时候，仅仅按照量约束下的在线分配策略进行决策有可能浪费掉了很多本来可以卖得更贵的流量

  • 起源

    • 搜索广告

      • 上下文广告

  • 广义第二高价

  • 广告网络（ADN）

  • 实时竞价

    • 供给方：广告交易平台（ADX）

    • 需求方平台（DSP）

      • 程序化交易

总结

  • 定向技术

    • 固定位置合约-》 受众定向

  • 交易形式

    • 合约交易 -》 竞价交易 -》 实时竞价交易

  • 产品展示逻辑

    • 展示广告 -》 原生广告

泛广告商业产品

团购

  • 特点

    • 其特殊性在于广告主除了付推广费用外，还向用户让利以获得转化

  • 主要广告主

    • 本地化的商铺

      • 获取新用户

游戏联运

  • 定义

    • 游戏联运根据用户的最终游戏内消费在推广渠道和游戏开发商之间分成的商业产品
